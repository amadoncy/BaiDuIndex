## 📖 项目概述

这是一个功能强大、界面友好的百度指数数据采集与分析平台，专为市场研究、舆情监测和竞品分析设计。系统支持多地区、多关键词的数据采集，提供数据可视化、人群画像分析和竞品分析，并生成专业数据报告，同时具备完善的用户管理系统。

## ✨ 核心功能

- **全方位数据采集**
  - 支持全国、区域、省份、城市级别的精细化数据采集
  - 自动登录百度指数平台并高效爬取数据
  - 支持多关键词批量数据处理
  - 实现断点续传，自动补充缺失数据

- **深度数据分析**
  - 关键词搜索趋势分析与可视化
  - 用户人群画像分析（年龄、性别、地域、兴趣分布）
  - 竞品分析与市场定位
  - 搜索需求图谱分析

- **专业报告生成**
  - 生成包含数据分析与市场洞察的HTML报告
  - 自定义报告内容和格式
  - 支持多种报告类型（完整报告、趋势分析、人群画像、需求分析）

- **用户体验优化**
  - 直观美观的图形界面
  - 多主题支持，可自定义界面风格
  - 搜索进度实时显示
  - 本地天气信息集成
  - 完善的用户管理系统（注册、登录、密码重置）

## 🔧 系统要求

- **操作系统**：Windows 7/10/11, macOS, Linux
- **Python环境**：Python 3.7+
- **浏览器**：Chrome（用于自动登录与数据爬取）
- **数据库**：MySQL 5.7+
- **最低配置**：4GB RAM, 100MB磁盘空间

## 📦 快速安装

1. **克隆项目到本地**
   ```bash
   git clone https://github.com/yourusername/BaiDuIndex.git
   cd BaiDuIndex
   ```

2. **安装依赖包**
   ```bash
   pip install -r requirements.txt
   ```

3. **配置数据库连接**
   - 编辑 `config/database.py` 文件，填入您的MySQL连接信息

4. **初始化数据库**
   ```bash
   python -m utils.db_init
   ```

5. **启动应用**
   ```bash
   python main.py
   ```

## 🚀 使用指南

### 数据采集

1. **登录系统**
   - 使用已有账号登录或注册新账号
   - 系统支持安全的密码重置功能

2. **配置采集参数**
   - 选择区域范围（全国/区域/省份/城市）
   - 输入需要分析的关键词（支持多关键词，以英文逗号分隔）
   - 设置采集日期范围

3. **开始采集**
   - 点击"开始采集"按钮，系统会自动执行数据采集
   - 采集进度实时显示
   - 数据自动保存至本地数据库和Excel文件

### 数据分析

1. **趋势分析**
   - 查看关键词搜索指数的时间趋势图表
   - 支持多关键词对比分析
   - 自动识别峰值和低谷

2. **人群画像分析**
   - 年龄分布：了解不同年龄段用户对关键词的关注度
   - 性别分布：分析男女用户比例
   - 兴趣分布：洞察用户的兴趣偏好
   - 地域分布：分析不同地区的搜索热度（支持热力图和表格视图）

3. **竞品分析**
   - 添加竞争关键词进行对比
   - 生成竞品分析图表和市场定位
   - 提供竞争策略建议

4. **需求分析**
   - 分析与关键词相关的搜索需求
   - 生成需求图谱和关键词关联词分析

### 报告生成

1. **选择报告类型**
   - 完整报告：包含所有分析数据
   - 趋势分析报告：聚焦搜索趋势变化
   - 人群画像报告：重点分析用户特征
   - 需求分析报告：专注于搜索需求洞察

2. **自定义报告内容**
   - 选择包含的数据模块（摘要、图表、预测、建议）
   - 设置报告文件名和保存路径

3. **导出并查看**
   - 系统自动生成HTML格式报告
   - 可直接在浏览器中打开查看
   - 报告包含数据分析结果和专业建议

## 💻 技术栈

- **前端界面**：PyQt5 5.15.9 (图形界面)
- **数据采集**：Selenium 4.15.2 (网页自动化)
- **数据处理**：Pandas 2.2.3, NumPy 2.2.3
- **数据可视化**：PyEcharts 2.0.8, Matplotlib 3.10.1
- **数据分析**：TensorFlow 2.19.0, Scikit-learn 1.6.1 (趋势预测)
- **数据存储**：PyMySQL 1.1.0 (数据库连接)
- **其他工具**：Pillow 11.1.0 (图像处理), captcha 0.5.0 (验证码处理)

## 📊 使用案例

- **市场研究**：分析产品关键词搜索趋势，了解市场需求变化
- **竞品分析**：对比不同品牌关键词的搜索热度和用户画像
- **营销策略**：根据人群画像制定精准的营销策略和内容
